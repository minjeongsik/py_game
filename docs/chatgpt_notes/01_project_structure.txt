프로젝트명: Aether Trail (PyGame)
기술 스택: C# / .NET 8 / MonoGame

루트 폴더 구조
- AGENTS.md: 작업 규칙, 빌드/검증 규칙, 보고 형식
- README.md: 프로젝트 개요와 현재 구현 범위 설명
- Program.cs: 게임 실행 진입점
- Game1.cs: 전체 게임 루프, 상태 전환, UI, 월드/전투 흐름 제어
- PyGame.csproj / PyGame.sln: 프로젝트 및 솔루션 파일

주요 소스 폴더
- Core
  - 게임 상태 관리, 카메라, 입력, 세션, 오디오 등 공통 시스템
  - 예: GameStateManager, GameSession, InputState, Camera2D
- World
  - 맵 데이터 로드, 월드 렌더링, 플레이어 이동, 포탈, 랜덤 인카운터 처리
  - 예: WorldMap, WorldMapLoader, WorldRenderer, PlayerController, EncounterService
- Battle
  - 턴 기반 1대1 전투 흐름과 메뉴 처리
  - 예: BattleController
- Creatures
  - 크리처 종족 정의와 개체 데이터
- Data
  - 게임 데이터 로딩 및 저장
  - 예: GameContentDatabase, SaveGameService, SaveGameData, MoveDefinition, ItemDefinition, ZoneDefinition
- UI
  - 픽셀 폰트 렌더링, UI 텍스트/오버레이 표시

콘텐츠 데이터 폴더
- Content/Data/creatures.json: 크리처 종 정의
- Content/Data/moves.json: 기술 정의
- Content/Data/items.json: 아이템 정의
- Content/Data/zones.json: 지역/인카운터 정의
- Content/Data/Maps/*.json: 맵 타일, 스폰 위치, 포탈 등 맵 데이터

현재 코드 구조 특징
- Game1.cs에 상태별 업데이트와 그리기 로직이 많이 모여 있음
- 실제 게임 데이터는 JSON 기반으로 분리되어 있음
- 월드 탐험 -> 랜덤 인카운터 -> 전투 -> 결과 반영 흐름이 연결되어 있음
- 저장/불러오기 기능이 이미 존재함
