Playable flow now supported
- Launch game
- Title screen shows `NEW GAME`
- `NEW GAME` creates a fresh session and enters `WorldState`
- Player moves one tile at a time with arrow keys or `WASD`
- Wall tiles block movement
- `Esc` opens pause
- `Resume` or `Esc` from pause returns cleanly to the same world session
